# Quick Start - LAUNCH AIMBOT NOW

## 🚀 FASTEST WAY TO RUN AIMBOT

### **Method 1: Double-Click (EASIEST)**

1. Find **`START-AIMBOT.bat`** in your repository folder
2. **Double-click it** (no need for "Run as Administrator" - it does it automatically)
3. Wait ~10-15 seconds for the build
4. **Aimbot launches automatically!** 🌟

---

### **Method 2: Right-Click Method**

1. Find **`LAUNCH-AIMBOT.bat`** in your repository folder
2. **Right-click** on it
3. Select **"Run as Administrator"**
4. Wait for build to complete
5. **Aimbot launches!** 🚀

---

### **Method 3: Create Desktop Shortcut (RECOMMENDED)**

1. Right-click **`START-AIMBOT.bat`**
2. Select **"Send to"** → **"Desktop (create shortcut)"**
3. Now you have a desktop shortcut!
4. Just **double-click the shortcut** anytime to launch Aimbot
5. **It will automatically:**
   - ✅ Request Admin privileges
   - ✅ Build the project
   - ✅ Launch Aimbot

## ⚠️ If Something Goes Wrong

### **"Admin privileges required"**
- Right-click → "Run as Administrator"

### **"Visual Studio not found"**
- Download: https://visualstudio.microsoft.com/downloads/
- Install with ".NET desktop development" selected
- Restart PC
- Try again

### **"Build failed"**
1. Open your repository folder
2. Delete these folders:
   - `bin/`
   - `obj/`
   - `WindowsFormsApp1/bin/`
   - `WindowsFormsApp1/obj/`
3. Try the launcher again

### **"Executable not found after build"**
- Open `AimbotScanner.sln` in Visual Studio
- Press `Ctrl+Shift+B` to build
- Wait for "Build succeeded" message
- Try launcher again
